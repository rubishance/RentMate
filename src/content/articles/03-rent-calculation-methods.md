# How to Calculate Rent Adjustments: Step-by-Step Guide
# כיצד לחשב עדכוני שכר דירה: מדריך שלב אחר שלב

## English

### The Basic Formula

The standard formula for CPI-linked rent adjustment is:

**New Rent = Base Rent × (Current Index ÷ Base Index)**

Let's break this down with a real example.

### Example Calculation

**Contract Details:**
- Base Rent: ₪4,000
- Contract Start Date: January 1, 2024
- Base Index (December 2023): 112.5
- Current Index (December 2024): 116.2

**Calculation:**
```
New Rent = ₪4,000 × (116.2 ÷ 112.5)
New Rent = ₪4,000 × 1.0329
New Rent = ₪4,131.60
```

**Result:** Rent increases by ₪131.60 (3.29% increase)

### Method 1: Known Index (מדד ידוע)

This method uses the index published **15 days before** the rent payment date.

**When to use:** Most residential contracts  
**Advantage:** Predictable and timely  
**How to find:** Check CBS website for index published 2 weeks before rent is due

### Method 2: Begin Index (מדד בגין)

This method uses the index from the **contract start date** as the base.

**When to use:** Long-term commercial leases  
**Advantage:** Reflects cumulative inflation  
**How to calculate:** Compare current index to original contract index

### Step-by-Step Process

1. **Identify your base rent** (from original contract)
2. **Find base index date** (usually contract signing month)
3. **Look up base index value** on CBS website
4. **Determine current index** (based on contract terms)
5. **Apply the formula** (New Rent = Base × Current ÷ Base)
6. **Round appropriately** (usually to nearest shekel)

### Common Pitfalls

❌ **Using wrong month's index** - Always verify the exact date  
❌ **Forgetting negative linkage** - Check if contract allows rent decreases  
❌ **Rounding errors** - Use precise decimals until final step  
❌ **Outdated CBS data** - Always use official, updated figures

### Pro Tips

✅ Set calendar reminders for index publication dates  
✅ Keep a spreadsheet with historical index values  
✅ Document all calculations for tenant transparency  
✅ Use automated tools to eliminate human error

## עברית

### הנוסחה הבסיסית

הנוסחה הסטנדרטית להתאמת שכר דירה מוצמד למדד היא:

**שכר דירה חדש = שכר דירה בסיסי × (מדד נוכחי ÷ מדד בסיס)**

בואו נפרק זאת עם דוגמה אמיתית.

### דוגמה לחישוב

**פרטי החוזה:**
- שכר דירה בסיסי: ₪4,000
- תאריך תחילת חוזה: 1 בינואר 2024
- מדד בסיס (דצמבר 2023): 112.5
- מדד נוכחי (דצמבר 2024): 116.2

**חישוב:**
```
שכר דירה חדש = ₪4,000 × (116.2 ÷ 112.5)
שכר דירה חדש = ₪4,000 × 1.0329
שכר דירה חדש = ₪4,131.60
```

**תוצאה:** שכר הדירה עולה ב-₪131.60 (עלייה של 3.29%)

### שיטה 1: מדד ידוע

שיטה זו משתמשת במדד שפורסם **15 יום לפני** מועד תשלום השכירות.

**מתי להשתמש:** רוב חוזי המגורים  
**יתרון:** צפוי ובזמן  
**איך למצוא:** בדקו באתר הלמ"ס את המדד שפורסם שבועיים לפני מועד התשלום

### שיטה 2: מדד בגין

שיטה זו משתמשת במדד מ**תאריך תחילת החוזה** כבסיס.

**מתי להשתמש:** חוזי שכירות מסחריים ארוכי טווח  
**יתרון:** משקף אינפלציה מצטברת  
**איך לחשב:** השוו את המדד הנוכחי למדד החוזה המקורי

### תהליך שלב אחר שלב

1. **זהו את שכר הדירה הבסיסי** (מהחוזה המקורי)
2. **מצאו את תאריך מדד הבסיס** (בדרך כלל חודש חתימת החוזה)
3. **חפשו את ערך מדד הבסיס** באתר הלמ"ס
4. **קבעו את המדד הנוכחי** (על בסיס תנאי החוזה)
5. **החילו את הנוסחה** (שכר דירה חדש = בסיס × נוכחי ÷ בסיס)
6. **עגלו כראוי** (בדרך כלל לשקל הקרוב)

### מלכודות נפוצות

❌ **שימוש בממד של חודש לא נכון** - תמיד אמתו את התאריך המדויק  
❌ **שכחת הצמדה שלילית** - בדקו אם החוזה מאפשר ירידת שכר דירה  
❌ **טעויות עיגול** - השתמשו בעשרוניות מדויקות עד השלב הסופי  
❌ **נתוני למ"ס לא מעודכנים** - תמיד השתמשו בנתונים רשמיים ומעודכנים

### טיפים מקצועיים
