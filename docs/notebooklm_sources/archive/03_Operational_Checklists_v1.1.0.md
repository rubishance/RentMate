Do this *together* with the tenant on the day they get the keys.

### 1. Readings
- [ ] **Water Meter:** Photo + Number reading.
- [ ] **Electricity Meter:** Photo + Number reading.
- [ ] **Gas Meter:** Photo + Number reading.

### 2. Inventory & Inspection
- [ ] **AC Units:** Turn on each unit (Cold & Heat) to verify it works.
- [ ] **Keys:** Count keys given (Example: "2 sets of 3 keys").
- [ ] **Paint:** Document status (e.g., "Freshly painted white").
- [ ] **Damages:** Note any existing scratches on floors/doors (protects both sides).
- [ ] **Furniture:** List all items provided (e.g., "Samsung Fridge Model X, 1 Blue Sofa with minor stain on left arm").

### 3. Signatures
- [ ] Both parties sign this protocol document.
- [ ] Upload photo of protocol to RentMate App > "Documents".

Do this 2 weeks *before* the contract ends.

- [ ] **Paint:** Remind tenant of obligation to paint (if in contract).
- [ ] **Bills:** Ask for proof of final payments (Arnona, Water, Electric) up to the exit date.
- [ ] **Inspection:** Walk through to check for *new* damages (beyond normal wear and tear).
