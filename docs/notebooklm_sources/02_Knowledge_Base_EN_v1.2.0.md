# Getting Started with RentMate (v1.2.0)

## Sign Up and Login
- To sign up, click "Login" then "Sign Up"
- Enter your email and verify it via the sent link

## Adding Your First Property
1. Click "Properties" -> "Add Property"
2. Fill in address, property type, area, and rooms.
3. Upload photos or documents (optional).

## Advanced AI Features

### Smart AI Assistant
The AI chat is an active property management tool:
- **Document Organization**: Upload a file via the Paperclip button and tell the bot: "Save this receipt to the Maintenance folder for Herzl St property."
- **Linkage Calculations**: Ask: "My rent is 5,000 NIS linked to CPI from Jan 2024. How much should I charge today?" The bot fetches live CBS data and calculates the delta.
- **Maintenance History**: Ask: "When was the last time I fixed the AC in the Haifa apartment?"
- **Manual Logging**: Tell the bot: "Log a 200 NIS plumbing expense for today."

## Key Modules
- **Dashboard**: Financial overview, activity feed, and visual analytics.
- **Payments**: Track rent collection and automatic reminders.
- **Documents**: Secure hub with folder organization (Warranties, Utility Bills, etc.).
- **Calculators**: Professional index calculation and reconciliation reports.
