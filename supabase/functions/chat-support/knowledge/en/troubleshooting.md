# Common Troubleshooting

## Login Issues
**Can't log in:**
- Ensure internet is active
- Try refreshing the page (F5)
- Clear browser cache
- Try another browser (Chrome, Firefox, Safari)

**Forgot password:**
1. Click "Forgot Password" on the login screen
2. Enter your email address
3. Check your inbox (also spam folder)
4. Click the password reset link
5. Set a new password

**Not receiving reset email:**
- Check spam/junk folder
- Verify email is correct
- Try again after 5 minutes
- Contact support: support@rentmate.co.il

## Demo Mode
**What is it?** Demo mode activates when the system is not connected to the database.

**How to exit:**
- The system needs to be configured with Supabase keys
- If you're a regular user - contact support
- If you're a developer - check the `.env` file

## File Upload Issues
**File won't upload:**
- Maximum size: 5MB
- Supported formats: PDF, JPG, PNG, DOCX
- Ensure stable internet connection
- Try a smaller file

**"Upload Failed" error:**
- Check you have available storage (Free plan: 500MB)
- Try refreshing and trying again
- Try another browser

## Contract Scanning Issues (AI Scanner)
**Scan doesn't recognize text:**
- Ensure image is clear and not blurry
- Avoid shadows or wrinkled papers
- Light the document well
- Photograph straight (not at an angle)

**Incorrect details:**
- Manually check and correct fields
- System uses AI - ~95% accuracy
- Send feedback to improve the system

## Linkage Calculation Issues
**Index not updating:**
- System pulls data from CBS once daily
- If new index published today - wait until tomorrow
- Check base date is correct

**Calculation seems wrong:**
- Ensure you selected correct index type (general/base/average)
- Check base date
- Compare with official CBS calculator

## Performance Issues
**App is slow:**
- Clear browser cache
- Close other tabs
- Check internet speed
- Try another device

**Page won't load:**
- Refresh page (Ctrl+R)
- Clear cookies
- Try incognito mode
- Update browser to latest version

## Payment and Subscription Issues
**Can't upgrade:**
- Ensure credit card details are correct
- Check card has sufficient balance
- Try another card
- Contact support

**Charged twice:**
- Check email - one may be confirmation only
- Contact support with screenshot of charge
