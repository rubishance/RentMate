# תכונות RentMate

## הצמדה למדד (CPI Linkage)
**מה זה?** הצמדה למדד היא מנגנון שמעדכן את שכר הדירה בהתאם לעליית מדד המחירים לצרכן.

**איך זה עובד ב-RentMate:**
- המערכת מושכת אוטומטית את נתוני המדד מהלשכה המרכזית לסטטיסטיקה
- מחשבת את אחוז השינוי בין תאריך הבסיס לתאריך הנוכחי
- מציגה את השכירות המעודכנת
- יוצרת הודעות תשלום מוכנות לשליחה לשוכר

**סוגי מדדים נתמכים:**
- מדד המחירים לצרכן (כללי)
- מדד יסוד (לפי תאריך חוזה)
- מדד ממוצע

## מרכז מסמכים (Documents Hub)
**מה אפשר לאחסן:**
- שטרות בעלות (טאבו)
- חוזי שכירות חתומים
- תיעוד תחזוקה (קבלות, חשבוניות)
- ביטוחים
- היתרי בנייה
- חשבונות ארנונה, מים, חשמל

**אבטחה:**
- כל הקבצים מוצפנים
- גישה רק למשתמש המורשה
- גיבוי אוטומטי

**ארגון:**
- חיפוש לפי שם קובץ, תאריך, קטגוריה
- תגיות מותאמות אישית
- תצוגת גלריה או רשימה

## ניהול שוכרים
- **פרופילים מלאים**: שם, טלפון, אימייל, תעודת זהות
- **היסטוריית תשלומים**: מעקב אחר תשלומים שבוצעו ותשלומים חסרים
- **תקשורת**: שליחת הודעות ישירות דרך WhatsApp או SMS
- **הערות**: תיעוד אירועים חשובים (תלונות, בקשות תחזוקה)

## מחשבון מתקדם
**מצבי חישוב:**
1. **חישוב פשוט**: הצמדה בסיסית למדד
2. **התחשבנות מלאה**: כולל חובות, תשלומי יתר, הנחות
3. **פירוט חודשי**: פירוק לפי חודשים

**תכונות:**
- ייצוא ל-PDF
- שליחה ישירה לשוכר
- שמירת היסטוריה

## תזכורות ותשלומים
- **תזכורות אוטומטיות**: 7 ימים לפני מועד תשלום
- **התראות SMS/Email**: ניתן להגדיר בהגדרות
- **מעקב סטטוס**: שולם / ממתין / באיחור
- **קישורי תשלום**: אינטגרציה עם Bit, PayBox (בקרוב)

## דוחות ומיסוי
- **דוח שנתי**: סיכום הכנסות והוצאות
- **ייצוא לרו"ח**: פורמט Excel תואם
- **מסלול 10%**: חישוב אוטומטי של מס לפי מסלול 10% (תושבי ישראל)
- **ניכוי הוצאות**: תיעוד הוצאות מוכרות (ארנונה, ועד בית, תיקונים)
